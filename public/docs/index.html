Introduction

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Authentication

Request OTP

This endpoint will be used to request OTP for login.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/v1/request-otp" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"mobile_number\": 9876543210
}"

const url = new URL(
    "http://127.0.0.1:8000/api/v1/request-otp"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "mobile_number": 9876543210
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "message": "OTP sent successfully."
}

Job

List

This endpoint will be used to list all the jobs.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/v1/job/list?limit=10&offset=0&search=job+name+or+country+and+state" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://127.0.0.1:8000/api/v1/job/list"
);

const params = {
    "limit": "10",
    "offset": "0",
    "search": "job name or country and state",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "POST",
    headers,
}).then(response => response.json());

Example response (200):

    "message": "Jobs listed successfully."
